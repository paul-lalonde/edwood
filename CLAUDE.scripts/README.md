# CLAUDE.scripts

Per-branch quick scripts used during cleanroom sessions. Re-use these
before writing a new one for similar work.

| script | purpose |
|---|---|
| `sizereport.sh` | report added LOC (implementation vs test, gross vs net of blank/comment lines) for the unified-frame-spans feature against the pre-feature baseline `230d818^`. Pass an alternate base as `$1`. |
